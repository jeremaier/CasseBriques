    • LE JEU DU CASSE BRIQUE

Ce projet a été réalisé au cours de l'année de terminale, pour être présenté à l'épreuve d'ISN du bac
Les trois auteurs de ce jeu sont : Jérémy, François et Fabien
Notre projet était de créer un casse brique fonctionnel sur le thème de l'espace



Les règles du jeu sont simples :
Il faut détruire toutes les briques pour gagner la partie
Si la balle tombe dans le vide, une vie est perdue
Si les vies tombent à 0, la partie est perdue

Il faut appuyer sur la barre espace pour lancer la balle

BON JEU !
